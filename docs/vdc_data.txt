
VDC Registers:

+----------+-------+--------------------------------------------------+---------+
|          | Hex   |                                                  | Kernal  |
| Register | Value | Description                                      | Default |
+----------+-------+--------------------------------------------------+---------+
| 0        | $00   | Total number of horizontal character positions   | $7E     |
+----------+-------+--------------------------------------------------+---------+
| 1        | $01   | Number of visible horizontal character positions | $50     |
+----------+-------+--------------------------------------------------+---------+
| 2        | $02   | Horizontal sync position                         | $66     |
+----------+-------+--------------------------------------------------+---------+
| 3        | $03   | Horizontal and vertical sync width               | $49     |
+----------+-------+--------------------------------------------------+---------+
| 4        | $04   | Total number of screen rows                      | $20     |
+----------+-------+--------------------------------------------------+---------+
| 5        | $05   | Vertical fine adjustment                         | $00     |
+----------+-------+--------------------------------------------------+---------+
| 6        | $06   | Number of visible screen rows                    | $19     |
+----------+-------+--------------------------------------------------+---------+
| 7        | $07   | Vertical sync position                           | $1D     |
+----------+-------+--------------------------------------------------+---------+
| 8        | $08   | Interlace mode control register                  | $00     |
+----------+-------+--------------------------------------------------+---------+
| 9        | $09   | Number of scan lines per character               | $07     |
+----------+-------+--------------------------------------------------+---------+
| 10       | $0A   | Cursor mode control                              | $20     |
+----------+-------+--------------------------------------------------+---------+
| 11       | $0B   | Ending scan line for cursor                      | $07     |
+----------+-------+--------------------------------------------------+---------+
| 12       | $0C   | Screen memory starting address (high byte)       | $00     |
+----------+-------+--------------------------------------------------+---------+
| 13       | $0D   | Screen memory starting address (low byte)        | $00     |
+----------+-------+--------------------------------------------------+---------+
| 14       | $0E   | Cursor position address (high byte)              | $00     |
+----------+-------+--------------------------------------------------+---------+
| 15       | $0F   | Cursor position address (low byte)               | $00     |
+----------+-------+--------------------------------------------------+---------+
| 16       | $10   | Light pen vertical position                      |         |
+----------+-------+--------------------------------------------------+---------+
| 17       | $11   | Light pen horizontal position                    |         |
+----------+-------+--------------------------------------------------+---------+
| 18       | $12   | Current memory address (high byte)               |         |
+----------+-------+--------------------------------------------------+---------+
| 19       | $13   | Current memory address (low byte)                |         |
+----------+-------+--------------------------------------------------+---------+
| 20       | $14   | Attribute memory starting address (high byte)    | $08     |
+----------+-------+--------------------------------------------------+---------+
| 21       | $15   | Attribute memory starting address {low byte)     | $00     |
+----------+-------+--------------------------------------------------+---------+
| 22       | $16   | Character horizontal size control register       |         |
+----------+-------+--------------------------------------------------+---------+
| 23       | $17   | Character vertical size control register         | $08     |
+----------+-------+--------------------------------------------------+---------+
| 24       | $18   | Vertical smooth scrolling and control register   | $20     |
+----------+-------+--------------------------------------------------+---------+
| 25       | $19   | Horizontal smooth scrolling and control register | $40     |
+----------+-------+--------------------------------------------------+---------+
| 26       | $1A   | Foreground/background color register             | $F0     |
+----------+-------+--------------------------------------------------+---------+
| 27       | $1B   | Address increment per row                        | $00     |
+----------+-------+--------------------------------------------------+---------+
| 28       | $1C   | Character set address and memory type register   | $20     |
+----------+-------+--------------------------------------------------+---------+
| 29       | $1D   | Underline scan-line-position register            | $07     |
+----------+-------+--------------------------------------------------+---------+
| 30       | $1E   | Number of bytes for block write or copy          |         |
+----------+-------+--------------------------------------------------+---------+
| 31       | $1F   | Memory read/write register                       |         |
+----------+-------+--------------------------------------------------+---------+
| 32       | $20   | Block copy source address (high byte)            |         |
+----------+-------+--------------------------------------------------+---------+
| 33       | $21   | Block copy source address (low byte)             |         |
+----------+-------+--------------------------------------------------+---------+
| 34       | $22   | Beginning position for horizontal blanking       | $7D     |
+----------+-------+--------------------------------------------------+---------+
| 35       | $23   | Ending position for horizontal blanking          | $64     |
+----------+-------+--------------------------------------------------+---------+
| 36       | $24   | Number of memory refresh cycles per scan line    | $05     |
+----------+-------+--------------------------------------------------+---------+

| 7        | $07   | light cyan              |

| 8        | $08   | dark red                |

| 9        | $09   | light red               |
+----------+-------+-------------------------+
| 10       | $0A   | dark purple             |
+----------+-------+-------------------------+
